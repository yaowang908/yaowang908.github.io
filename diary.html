Docker学习

  • 使用docker run -v将主机上的目录挂载到容器上要在容器运行前，否则很麻烦。
  • docker run -v 在windows boot2docker 下绝对路径要已双斜线开头"//c/Users/DATA/volumn:/src/webapp"否则会出现绝对路径错误
  • 使用link参数连接数据库也要在容器运行前执行
  • 容器已经运行后，想要在内部执行命令使用 docker exec -it CONTAINER_NAME bash
